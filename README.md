# python-web-falcon-hello-world

## Description
Simple web app for a falcon project.
Uses gunicorn to serve app.

## Tech stack
- python
- falcon
- gunicorn

## Docker stack
- tiangolo/meinheld-gunicorn

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
